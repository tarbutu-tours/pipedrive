# שליחת דוח השוואה למייל כל 3 ימים

## הכנה (פעם אחת)

1. **הגדרת מייל**  
   העתק `config.example.json` ל-`config.json` ומלא:
   - `email.to` – הכתובת שלך
   - `email.smtp_user` – כתובת המייל שממנה נשלח (למשל Gmail)
   - `email.smtp_password` – סיסמת אפליקציה (ב-Gmail: הגדרות → אבטחה → סיסמאות אפליקציה)

2. **התקנת nodemailer** (אם עדיין לא מותקן):
   ```bash
   cd c:\Users\user\Documents\WORK
   npm install nodemailer
   ```

## תזמון ב-Windows (תכנון משימות)

1. פתח **תכנון משימות** (Task Scheduler).
2. **צור משימה בסיסית**:
   - שם: `דוח השוואת טיולים`
   - טריגר: **יומי**, חזור כל **3 ימים**.
   - פעולה: **הפעלת תוכנה**.
   - תוכנית/סקריפט: `node`
   - ארגומנטים: `competitor_research\send_report_email.cjs`
   - התחל ב: `c:\Users\user\Documents\WORK`

   או השתמש בקובץ הבאטץ:
   - תוכנית: `c:\Users\user\Documents\WORK\competitor_research\run_scan_and_email.bat`

3. שמור. בכל 3 ימים המערכת תריץ סריקה, תבנה את האקסל ותשלח לך את הקובץ במייל.

## הרצה ידנית

- **רק סריקה + אקסל (בלי מייל):**  
  דאבל-קליק על `run_scan.bat` או הרצה:  
  `node competitor_research/run_full_scan.cjs`

- **סריקה + שליחת מייל:**  
  `node competitor_research/send_report_email.cjs`  
  או דאבל-קליק על `run_scan_and_email.bat`.
