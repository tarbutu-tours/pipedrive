# איך להריץ את האפליקציה

---

## מומלץ – בלי Docker (רק Node.js)

**לא צריך Docker, לא WSL, לא PostgreSQL.** רק Node.js.

1. **התקן Node.js** (אם עדיין לא): https://nodejs.org — בחר גרסה **LTS**.
2. **לחץ פעמיים** על הקובץ:
   ```
   הפעל_בלי_Docker.bat
   ```
3. חכה: יתקינו חבילות, ייכון מסד נתונים, ויופעל שרת. כשמופיע **"Server listening"** – הדפדפן ייפתח אוטומטית עם **לינק כניסה ישירה** (בלי אימייל וסיסמה).
4. **כניסה:** לחץ על "כניסה לאפליקציה" או גלוש ישירות ל: http://localhost:3000/auth/enter

החיבור ל-**Pipedrive** נשמר מהקובץ `.env` – הכל יעבוד עם החשבון שלך.

---

## עם Docker (אם אתה משתמש ב-Docker)

1. **לחץ פעמיים** על:
   ```
   הפעל_את_האפליקציה.bat
   ```
2. חכה עד שיופיע "Server listening".
3. **לחץ פעמיים** על:
   ```
   פתח_את_האפליקציה_בדפדפן.bat
   ```
4. כניסה: לחץ על הלינק או גלוש ל־ http://localhost:3000/auth/enter

---

## עם טרמינל (אותה תוצאה)

1. **פתח טרמינל** בתיקיית הפרויקט:
   ```
   c:\Users\user\Documents\WORK\pipedrive-sales-ai
   ```

2. **הרץ:**
   ```bash
   docker-compose up --build
   ```
   (בפעם הראשונה זה יוריד תמונות ויבנה את הפרויקט – יכול לקחת כמה דקות.)

3. **חכה** עד שתופיע שורה כמו:
   ```
   Server listening at http://0.0.0.0:3000
   ```

4. **פתח בדפדפן:**  
   **http://localhost:3000**

5. **כניסה:** גלוש ל־ http://localhost:3000/auth/enter (או לחץ על "כניסה לאפליקציה" בדף ההתחברות).

אחרי הכניסה תיכנס לצ'אט. החיבור ל-**Pipedrive** (tarbutu) כבר מוגדר מהקובץ `.env` שלך – כל פעולה תעבוד עם החשבון שלך.

---

## טרמינל (בלי Docker)

אם אתה מעדיף טרמינל במקום קובץ bat:

1. התקן **Node.js** (גרסה 20+) מ־ https://nodejs.org
2. בתיקיית הפרויקט הרץ:
   ```bash
   הפעל_בלי_Docker.bat
   ```
   או ידנית:
   ```bash
   set DATABASE_URL=file:./prisma/dev.db
   npm install
   npx prisma generate
   npx prisma migrate deploy
   npx prisma db seed
   npm run dev
   ```
3. בדפדפן: http://localhost:3000 או כניסה ישירה: http://localhost:3000/auth/enter
